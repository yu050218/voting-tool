# 万能投票工具

面向校内选举、民主测评、作品投票的一体化投票平台原型。项目依据本文件夹中的《万能投票工具项目设计报告(1).pdf》开发，当前版本完成了前台参与、后台任务配置、账号生成、防重复提交、统计分析和 CSV 导出等核心流程。

## 一、功能概览

- 前台首页：展示已发布任务，支持搜索、查看详情、登录参与或免密投票。
- 校内选举：支持候选对象展示、匿名投票、最多可选数量限制、得票排名。
- 民主测评：支持多个对象共用同一套题目，支持评分题、等级评价题和维度统计。
- 作品投票：支持作品简介、作者信息、附件名称、单选/多选投票和排名。
- 后台管理：支持新建任务、编辑基本信息、配置题目与对象、发布、取消和删除。
- 账号管理：支持随机账号生成、账号 CSV 下载、账号提交状态记录。
- 数据统计：支持应参与人数、已完成人数、完成率、得票数、测评总分、维度均分。
- 数据导出：统计结果和账号列表均可导出为 CSV，Excel 可直接打开。

## 二、技术栈

- Vite
- React
- lucide-react
- localStorage 本地数据存储

当前版本是适合课程实训和演示的前端原型，不依赖服务器和数据库。后续可以按 README 末尾的建议接入真实后端。

## 六、使用流程

1. 进入前台首页，查看已发布任务。
2. 免密任务点击“开始投票”，需要登录的任务点击“登录参与”。
3. 完成选举、测评或作品投票后提交。
4. 进入后台管理，查看任务列表。
5. 点击“编辑”修改任务基本信息。
6. 点击“配置”添加题目、候选人、测评对象或作品，并生成账号。
7. 点击“统计”查看排名、完成率和维度结果。
8. 使用“导出 CSV”保存统计结果。

## 七、数据说明

当前数据保存在浏览器 localStorage 中，刷新页面不会丢失。同一浏览器内多次演示会保留上次提交记录。

如需恢复初始数据，可以在浏览器开发者工具中清除 localStorage，或在控制台执行：

```js
localStorage.removeItem("万能投票工具-任务数据-v1")
location.reload()
```
